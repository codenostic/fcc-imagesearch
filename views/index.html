Image Search Abstraction layer

User stories:
  1. Get list of image search results(image url, alt text, page url) for a query string
  2. Get list of latest searches
  3. Paginate your query by passing a paramater offset=number

Example usage:

1) Image Search Query : https://fcc-imagesearch-api.glitch.me/api/imagesearch?q=cat&offset=2
2) Latest Search Strings: https://fcc-imagesearch-api.glitch.me/api/latest/imagesearch
